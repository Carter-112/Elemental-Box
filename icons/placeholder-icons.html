ElementalBox Icon Generator

Instructions:

This tool generates app icons for ElementalBox PWA from the SVG template.

  1. Click the "Generate Icons" button below
  2. Right-click on each generated icon and select "Save Image As..."
  3. Save the 192x192 image as "icon-192x192.png" in the icons folder
  4. Save the 512x512 image as "icon-512x512.png" in the icons folder
